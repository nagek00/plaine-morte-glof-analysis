\section{Methods}
\label{sec:methods}

\subsection{Series Construction and GLOF Masking}

Annual block maxima are extracted from each daily discharge record. Three annual maximum
series (AMS) are constructed: (i)~the \textbf{full} series spanning the complete record,
(ii)~a \textbf{GLOF-masked} series in which values within $\pm 5$ days of each confirmed
GLOF date are set to missing before the block maximum is taken, and (iii)~a
\textbf{reference} series restricted to the pre-GLOF period (1944--2010). The five-day
symmetric window captures the typical GLOF recession limb at this gauge. Hypothesis tests
are applied exclusively to the GLOF period 2011--2018; post-2018 years are excluded
because the bypass channel altered the outburst mechanism.

\subsection{GEV Flood Frequency Analysis}

A Generalised Extreme Value (GEV) distribution is fitted to each AMS by maximum
likelihood estimation. The GEV encompasses Gumbel ($\xi = 0$), Fr\'{e}chet ($\xi > 0$),
and Weibull ($\xi < 0$) families, allowing the data to determine the appropriate tail
behaviour. Design quantiles for $T = 2, 5, 10, 20, 50, 100, 200$\,yr are derived from
the fitted parameters with delta-method confidence intervals. Gringorten plotting
positions are used for empirical return periods, and model fit is compared using the
Akaike Information Criterion.

\subsection{Homogeneity Tests}
\label{sec:methods:tests}

Two complementary tests compare the 2011--2018 annual maxima against the pre-GLOF
reference, applied identically to both datasets.

\textbf{KS test}: each 2011--2018 peak is transformed to a uniform variate via the
reference GEV CDF (probability integral transform). The Kolmogorov--Smirnov statistic
tests departure from Uniform(0,1) with critical value
$C_\alpha = K_\alpha / (\sqrt{n} + 0.12 + 0.11/\sqrt{n})$, $K_{0.05} = 1.358$
(Module~1, Eq.~3.3). Tests run separately for GLOF-inclusive (Group~A, $n = 8$) and
GLOF-masked (Group~B, $n = 8$) peaks.

\textbf{Rank-Sum test}: the non-parametric Wilcoxon Rank-Sum test (Module~1, \S2.4.1)
compares (5a) GLOF-year full peaks vs.\ non-GLOF years, (5b) GLOF-year masked peaks
vs.\ non-GLOF years, and (5c) pre-GLOF reference vs.\ masked GLOF-era peaks as a
secular change check.
